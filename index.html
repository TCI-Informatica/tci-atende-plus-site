Política de Privacidade TCI Atende+

TCI Informática

Última atualização: 12/02/2026

A TCI Informática respeita a privacidade e a proteção de dados pessoais. Esta Política de Privacidade descreve como o aplicativo TCI Atende+ trata informações necessárias para suas operações.

Este aplicativo é destinado exclusivamente a profissionais autorizados vinculados às operações da TCI.

1. Introdução

Esta Política explica quais dados podem ser tratados, as finalidades, como armazenamos as informações e como você pode entrar em contato com a TCI para exercer seus direitos previstos na Lei nº 13.709/2018 (LGPD).

2. Dados Coletados

O aplicativo coleta apenas os dados necessários para o funcionamento operacional do sistema, incluindo:

2.1 Dados de Identificação

  • Identificação funcional do usuário
  • Credenciais de acesso

2.2 Dados Operacionais

  • Informações relacionadas às fichas de atendimento
  • Registros de horários
  • Quilometragem
  • Dados inseridos manualmente pelo usuário durante o atendimento

2.3 Dados Técnicos

  • Informações necessárias para autenticação
  • Registros de sincronização com servidores da TCI

3. Dados que NÃO são coletados

Na versão atual, o aplicativo não coleta:

  • Dados de localização automática (GPS)
  • Imagens ou arquivos da câmera do dispositivo
  • Dados biométricos
  • Dados financeiros
  • Informações para fins publicitários

Observação: o aplicativo também não utiliza notificações push na versão atual.

4. Finalidade do Tratamento

Os dados são utilizados exclusivamente para:

  • Registro e gerenciamento de atendimentos
  • Sincronização com sistemas internos da TCI
  • Controle operacional e auditoria
  • Garantia de segurança e integridade das informações

5. Armazenamento e Segurança

Para permitir funcionamento offline, alguns dados podem ser armazenados temporariamente no dispositivo. Quando houver conectividade, os dados são sincronizados com servidores mantidos pela TCI.

A TCI adota medidas técnicas e administrativas adequadas para proteger as informações contra acesso não autorizado, alteração ou divulgação indevida.

6. Compartilhamento de Dados

A TCI não comercializa dados pessoais. O compartilhamento poderá ocorrer apenas quando necessário para:

  • Cumprimento de obrigação legal
  • Determinação de autoridade competente
  • Execução das atividades operacionais da própria TCI

7. Retenção

Os dados são mantidos pelo período necessário para cumprir as finalidades operacionais, atender exigências legais e manter registros históricos conforme regulamentação aplicável.

8. Direitos do Titular

Nos termos da LGPD, o titular poderá solicitar:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados
  • Eliminação quando aplicável
